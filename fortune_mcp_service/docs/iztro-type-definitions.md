# iztro 类型定义文档

## 概述
iztro库是一个用于紫微斗数（Purple Star Astrology）的JavaScript库，提供完整的TypeScript类型定义。

## 主要类型分类

### 1. 国际化类型 (International Types)
- 支持多语言：简体中文、繁体中文、英文、韩文、日文
- 灵活的输入：可以使用不同语言变体输入天文术语
- 可配置的语言输出

### 2. 通用类型 (Common Types)
- 基础数据类型定义
- 通用工具类型

### 3. 星曜类型 (Star Types)
- 主星、辅星、杂曜等各类星曜的类型定义
- 星曜属性：亮度、四化等

### 4. 宫位类型 (Palace Types)
- 十二宫位的类型定义
- 宫位属性和状态

### 5. 运势类型 (Horoscope Types)
- 大运、流年、流月等运势相关类型
- 运势分析结果类型

### 6. 星盘类型 (Astrolabe Types)
- 完整星盘对象的类型定义
- 星盘方法和属性

### 7. 配置和插件类型 (Configuration and Plugin Types)
- 全局配置选项
- 第三方插件接口

## 特色功能
1. **类型安全**：提供完整的TypeScript类型定义
2. **多语言支持**：灵活的国际化选项
3. **可扩展性**：支持插件和配置扩展
4. **结构化数据**：复杂的占星学计算结构化处理

## 用途
为JavaScript/TypeScript应用程序提供结构化、类型安全的紫微斗数数据处理方式。
